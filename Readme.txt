EMC bit installator

for WIN10

WARNING: running this script will change your windows settings!

- this script automatically installs the Burn In Test software from Passmark
- creates shotrtcuts for specific tests being used for EMC testing
- changes all of the connected IP addresses to be static
- changes windows settings by running the bash script /bit/Windows_setup.bat
- installs drivers for testing tools by running the bash script /bit/Install_drivers.bat
- creates the report and reboots the computer
